# Garbage Talk
just a garbage talk machine

## Features
- choose the person you want to talking to
- create a garbage sentence


### Environment SetUp - 環境建置
Node.js

### Installing - 專案安裝流程
打開你的 terminal，Clone 此專案至本機電腦
```
git clone https://github.com/LeonardoRoosevelt/garbage_talk
```
開啟終端機(Terminal)，進入存放此專案的資料夾
```
cd garbage_talk
```
安裝 npm 套件
```
在 Terminal 輸入 npm install 指令
```
安裝 nodemon 套件
```
在 Terminal 輸入 nodemon app.js 指令
```
啟動伺服器，執行 app.js 檔案
```
nodemon app.js
```
當 terminal 出現以下字樣，表示伺服器與資料庫已啟動並成功連結
```
The Express server is running on http://localhost:3000
```

現在，你可開啟任一瀏覽器瀏覽器輸入 "http://localhost:3000" 開始使用
